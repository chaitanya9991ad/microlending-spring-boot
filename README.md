Microlending System

The Microlending System is a backend application built using Java, Spring Boot, Spring Security, JWT, Hibernate, and MySQL. It simplifies the loan lifecycle by allowing users to register, authenticate securely, apply for micro-loans, track loan status, and make installment payments, while administrators manage loan approvals and monitor repayments.

Features
Secure user authentication and authorization using JWT.
Role-based access control for Admin and User.
User registration and login.
Loan application and approval workflow.
Installment (EMI) generation and repayment tracking.
Loan status management (Pending, Approved, Rejected, Closed).
Payment verification with dummy payment integration.
Email notifications for loan approval and completion.
RESTful APIs tested using Postman.
Database management using MySQL with Spring Data JPA.
